Panduan Menggunakan Script di Termux JNCUK

Langkah 1: Download dan Ekstrak Script Termux

  • Download Script ZIP: Download file script Termux dalam bentuk ZIP di WEBSITE JNCUK.
  • Ekstrak ZIP ke Penyimpanan Internal HP: Ekstrak file ZIP ke folder yang mudah diakses di penyimpanan internal HP Anda, misalnya: 
    /sdcard/download/script

Langkah 2: Persiapan di Termux

  • Buka Aplikasi Termux: Jika belum terinstal, unduh dan install aplikasi Termux dari Google Play Store atau F-Droid.
  • Masuk ke Folder Script: Buka Termux dan arahkan ke folder tempat Anda mengekstrak file ZIP dengan mengetikkan perintah berikut: 
    cd /sdcard/download/script
  • Verifikasi Isi Folder: Ketik perintah ls untuk memastikan semua file yang diperlukan ada di dalam folder: 
    ls

Langkah 3: Install Library yang Dibutuhkan

  • Update Paket di Termux: Perbarui paket-paket di Termux dengan perintah berikut: 
    pkg update && pkg upgrade
  • Install Node.js: Install Node.js yang diperlukan untuk menjalankan skrip dengan perintah: 
    pkg install nodejs
  • Install Axios: Install Axios dan library lainnya yang dibutuhkan dengan perintah: 
    npm install axios

Langkah 4: Simpan Kukis (Login ke KlikIndomaret)

  • Login ke KlikIndomaret dengan Skrip masuk.js: Setelah semua library terpasang, jalankan skrip masuk.js untuk login ke KlikIndomaret dan menyimpan token atau kukis: 
    node masuk.js
  • Masukkan Nomor HP dan Password: Ikuti petunjuk untuk memasukkan nomor HP dan password Anda saat diminta.
  • Simpan Token/Kukis: Jika login berhasil, token atau kukis akan disimpan di file kuki.txt.

Langkah 5: Cek Stok Produk

  • Jalankan Skrip untuk Cek Stok: Setelah login berhasil dan kukis disimpan, Anda bisa mengecek stok produk dengan menjalankan skrip main.js: 
    node main.js
  • Masukkan Kode Toko dan PLU: Ikuti petunjuk untuk memasukkan Kode Toko dan PLU produk yang ingin Anda cek.
Catatan:
  • Pastikan Anda telah mengikuti setiap langkah dengan benar, terutama saat login dan menyimpan kukis, karena kukis tersebut digunakan untuk memverifikasi akses Anda ke KlikIndomaret.
  • Jika ada masalah saat menjalankan skrip atau login, pastikan Anda sudah memperbarui paket-paket di Termux dan menginstall semua dependensi yang dibutuhkan.
